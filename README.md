# Ravishka JOB Board

A modern, responsive, and fully-featured job board application built for software students and professionals. Developed with Next.js 16, React 19, and TypeScript, this platform provides a seamless experience for browsing, searching, and viewing job opportunities.

## 🚀 Key Features

*   **Modern Landing Page:** A beautifully designed, mobile-responsive landing page with a professional purple-themed aesthetic.
*   **Comprehensive Job Search & Filtering:** Powerful filtering system allowing users to search jobs by:
    *   Job Title
    *   Company Name
    *   Location
    *   Job Type (Full-time, Part-time, Contract, etc.)
    *   Required Skills
*   **Dynamic Job Details Pages:** Individual, detailed pages for each job listing.
*   **Human-Readable URLs:** Clean, SEO-friendly slugs generated from job titles and company names with conflict-aware numbering (e.g., `/jobs/frontend-developer-techcorp-1`).
*   **Pagination:** Smooth and efficient pagination for handling large numbers of job listings.
*   **Responsive UI:** Carefully crafted with CSS Modules to look great on all devices, from mobile phones to large desktop screens.

## 🛠️ Technology Stack

*   **Framework:** [Next.js 16](https://nextjs.org/) (Using the `app` router)
*   **Library:** [React 19](https://react.dev/)
*   **Language:** [TypeScript](https://www.typescriptlang.org/) for robust, type-safe code.
*   **Styling:** Vanilla CSS Modules for scoped, maintainable styles.
*   **Icons & Fonts:** Optimized loading using `next/font`.

## 💻 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

*   Node.js (v18.17 or higher recommended)
*   npm, yarn, pnpm, or bun package manager

### Installation

1.  Clone the repository or download the source code.
2.  Navigate into the project directory:
    ```bash
    cd job-board
    ```
3.  Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

### Running the Development Server

Start the development server with:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application running. You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## 🏗️ Architecture & Development Notes

*   **Routing:** The application heavily utilizes the Next.js App Directory structure (`app/`).
*   **Slug Generation Strategy:** Job IDs are abstracted behind clean URLs. Slugs are dynamically generated by combining the job name and company name, replacing spaces with hyphens, and converting to lowercase. A collision handling system appends numbers to ensure unique URLs.
*   **Component Modularity:** UI elements are broken down into self-contained components in the `/components` folder, each paired with its own `.module.css` file to prevent style leakage and maintain a clean global scope.
